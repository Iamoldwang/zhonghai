- 社区：社区公告，信息，活动，等等(后期可以加入团购，业主广场聊天室，周边服务等等)
- 管家：便民服务，保洁，保姆，洗衣，租房卖房，车险，快递，代驾，车位，洗车，二手物品交易等等
